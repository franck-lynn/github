# 配置提交的 config 文件

在 .git/config 文件中, 设置如下

```config
[core]
	repositoryformatversion = 0
	filemode = false
	bare = false
	logallrefupdates = true
	symlinks = false
	ignorecase = true
[remote "github"]
	url = git@github.com:franck-lynn/github.git
	url = git@gitee.com:franck-lynn/github.git
	fetch = +refs/heads/*:refs/remotes/github/*
[pull]
	rebase = false
[branch "master"]
	remote = github
	merge = refs/heads/master

```
